What is Wardle Bot

Compete in monthly Wordle leagues with friends using the most fun Telegram bot.

This Telegram bot transforms any group into a Wordle competition, with a monthly ranking updated daily, prizes for the best players, NPCs and much more.

Main Features of Wardle Bot

üèÜ Monthly player ranking

Whenever a chat member shares their game, they receive a score ranging from 0 (if they didn't find the word) to 6 (if they found it on the first try). This creates a ranking with all chat members. At the end of each month, prizes are distributed to members who finished in the top 3 and a new league begins with scores reset to 0.

/clasificacion - Check the ranking

ü•á Virtual prize and trophy system

On the last day of the month, trophies are distributed to the three best players. Trophies vary depending on the current league, and are independent between different leagues - meaning if you participate in multiple leagues, you only have trophies within the chat of the league where you won them.

/premios - Check chat trophies

üé≠ Famous characters

You can add famous characters who will play autonomously each day and score more or less depending on their skill - different characters have different skill levels, some will be very difficult to beat while others will always rank lower in the standings.

/agregar-personaje - Add characters to the game

/eliminar-personaje - Remove characters from the game

üåç Global ranking

At any time you can check the top 3 of the current league among all participants from all chats.

/top - Show global top 3

üòä Automatic reactions

When a game is shared, the bot reacts with an emoji to the message that varies depending on how good or bad the game was to provide visual confirmation that the game has been registered correctly.

Available Wardle Bot Commands

  • /clasificacion - Show current league ranking
  • /leyenda - Show scoring table
  • /agregar-personaje - Options to add characters to the game
  • /eliminar-personaje - Options to remove a character from the game
  • /premios - Check virtual trophies won and those at stake
  • /top - Show top 3 players from all leagues

Data Usage and Privacy

Required permissions

When adding Wardle Bot to a group, it is given access to all messages shared from that moment on. If this weren't the case, it couldn't react or save the games shared with it.

Privacy commitment

No message is consulted or saved unless it's a shared game or a bot command.
